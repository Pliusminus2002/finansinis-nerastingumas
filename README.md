# Finansinis neraštingumas

Studentams skirta finansų valdymo web programėlė.

## Funkcijos

- Pajamų ir išlaidų įvedimas
- Išlaidų kategorijos
- Mėnesio biudžeto planavimas
- Taupymo tikslas
- Grafikai ir apžvalga
- Finansiniai patarimai
- Veikia telefone kaip web app / PWA
- Duomenys saugomi naršyklėje per localStorage

## Paleidimas

```bash
npm install
npm run dev
```

Tada atidarykite naršyklėje adresą, kurį parodo terminalas, pvz.:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Publikavimas

Projektą galima įkelti į GitHub ir deployinti per Vercel.
